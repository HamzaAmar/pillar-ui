# Shared CSS Utility Classes in Pillar

Pillar focuses on **small bundle size**, **high performance**, and **easy-to-read** utility CSS. Our naming convention groups classes by file, property, and modifier so you can quickly apply spacing, alignment, sizing, and more without writing custom CSS.

## Alignment Utilities

Easily control flex and grid alignment in any browser.

| Class             | CSS Property                      | Description                               |
| ----------------- | --------------------------------- | ----------------------------------------- |
| `Ajc-start`       | `justify-content: flex-start;`    | Align items at the start                  |
| `Ajc-center`      | `justify-content: center;`        | Center items horizontally                 |
| `Ajc-end`         | `justify-content: flex-end;`      | Align items at the end                    |
| `Ajc-between`     | `justify-content: space-between;` | Distribute items with space between       |
| `Ajc-around`      | `justify-content: space-around;`  | Distribute items with space around        |
| `Ajc-evenly`      | `justify-content: space-evenly;`  | Distribute items with equal space         |
| `Aji-start`       | `align-items: flex-start;`        | Align items at the start (cross axis)     |
| `Aji-center`      | `align-items: center;`            | Center items (cross axis)                 |
| `Aji-end`         | `align-items: flex-end;`          | Align items at the end (cross axis)       |
| `Aji-baseline`    | `align-items: baseline;`          | Align items along the baseline            |
| `Aji-stretch`     | `align-items: stretch;`           | Stretch items to fill the container       |
| `Aac-start`       | `align-content: flex-start;`      | Align wrapped lines at the start          |
| `Aac-center`      | `align-content: center;`          | Center wrapped lines horizontally         |
| `Aac-end`         | `align-content: flex-end;`        | Align wrapped lines at the end            |
| `Aac-between`     | `align-content: space-between;`   | Distribute wrapped lines with gaps        |
| `Aac-around`      | `align-content: space-around;`    | Distribute wrapped lines with gaps around |
| `Aac-evenly`      | `align-content: space-evenly;`    | Evenly distribute wrapped lines           |
| `Ajc-self-start`  | `align-self: flex-start;`         | Override self-alignment to start          |
| `Ajc-self-center` | `align-self: center;`             | Override self-alignment to center         |
| `Ajc-self-end`    | `align-self: flex-end;`           | Override self-alignment to end            |

## Sizing Utilities

Set width, height, and aspect ratios without writing CSS.

### Width

| Class       | CSS Rule              | Description                    |
| ----------- | --------------------- | ------------------------------ |
| `Sw-25`     | `width: 25%;`         | Quarter of parent width        |
| `Sw-50`     | `width: 50%;`         | Half of parent width           |
| `Sw-75`     | `width: 75%;`         | Three-quarters of parent width |
| `Sw-100`    | `width: 100%;`        | Full parent width              |
| `Sw-128p`   | `width: 128px;`       | Fixed width in pixels          |
| `Sw-256p`   | `width: 256px;`       | Larger fixed width in pixels   |
| `Sw-fit`    | `width: fit-content;` | Shrink to fit content          |
| `Sw-screen` | `width: 100vw;`       | Full viewport width            |

### Height

| Class       | CSS Rule               | Description                     |
| ----------- | ---------------------- | ------------------------------- |
| `Sh-25`     | `height: 25%;`         | Quarter of parent height        |
| `Sh-50`     | `height: 50%;`         | Half of parent height           |
| `Sh-75`     | `height: 75%;`         | Three-quarters of parent height |
| `Sh-100`    | `height: 100%;`        | Full parent height              |
| `Sh-128p`   | `height: 128px;`       | Fixed height in pixels          |
| `Sh-384p`   | `height: 384px;`       | Large fixed height in pixels    |
| `Sh-fit`    | `height: fit-content;` | Shrink to fit content           |
| `Sh-screen` | `height: 100vh;`       | Full viewport height            |

## 5. Positioning Utilities

Control positioning and insets quickly.

| Class             | CSS Properties                                           | Description                             |
| ----------------- | -------------------------------------------------------- | --------------------------------------- |
| `P-relative`      | `position: relative;`                                    | Establish a containing block            |
| `P-absolute`      | `position: absolute;`                                    | Remove from flow, position via inset    |
| `P-fixed`         | `position: fixed;`                                       | Fixed relative to viewport              |
| `P-sticky`        | `position: sticky;`                                      | Stick within parent at threshold        |
| `P-start-start`   | `top: 0; left: 0;`                                       | Pin to top-left corner                  |
| `P-start-center`  | `top: 0; left: 50%; transform: translateX(-50%);`        | Pin to top and center horizontally      |
| `P-start-end`     | `top: 0; right: 0;`                                      | Pin to top-right corner                 |
| `P-center-start`  | `top: 50%; left: 0; transform: translateY(-50%);`        | Pin to vertical center and left         |
| `P-center-center` | `top: 50%; left: 50%; transform: translate(-50%, -50%);` | Center both vertically and horizontally |
| `P-center-end`    | `top: 50%; right: 0; transform: translateY(-50%);`       | Pin to vertical center and right        |
| `P-end-start`     | `bottom: 0; left: 0;`                                    | Pin to bottom-left corner               |
| `P-end-center`    | `bottom: 0; left: 50%; transform: translateX(-50%);`     | Pin to bottom and center horizontally   |
| `P-end-end`       | `bottom: 0; right: 0;`                                   | Pin to bottom-right corner              |
