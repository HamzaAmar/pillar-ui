## Usage

```tsx
import { Alert } from '@pillar-ui/core'
```

```tsx
<Alert title="Title" description="Description" icon={<AlertCircle />} />
```

## Examples

### Content

The Content is not a props but we use it here to describe all the props related to the content of the alert.

- **title**: The title of the alert.
- **description**: The description of the alert.
- **icon**: The icon of the alert.

### Closable

The `closable` prop controls whether the alert can be closed by the user. When set to `true`, a close button appears on the alert, allowing users to dismiss it. If not specified, it defaults to `false`.
