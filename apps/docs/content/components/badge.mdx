### variant

The variant prop determines the type of Badge to render. It accepts the following values:

- **numeric**: displays a numeric value.
- **icon**: displays an icon.
- **dot**: displays a dot.

<BadgePlayGround.Variants />

```tsx showLineNumbers title="Badge.tsx"

  return (
    <div className="l_flow_md">

        <Badge variant="numeric" number={15} max={15} color="danger" />
        <Badge variant="icon" icon={<Check />} color="danger" />
      </Flex>
    </div>
  )
}
```

### max?: number

The max prop is required for the numeric variant. It determines the maximum value that can be displayed. If the number prop is greater than max, the badge will display max+.

## Use Case

Badges are often used to highlight important or urgent information, such as unread messages, new notifications, or error messages. They can also be used to display a user's status, such as "online" or "away."

## Implementation details

The Badge component typically consists of a small shape, such as a circle or square, with a number or text inside. The number or text is often used to indicate the quantity of items or the specific type of notification. The Badge component can be implemented using HTML and CSS, or with a UI framework or library such as React, Angular, or Vue.

## Accessibility

When implementing the Badge component, it's important to ensure that it is accessible to all users, including those with disabilities. This can include using appropriate colors and contrast ratios, providing alternative text for screen readers, and ensuring that the component is keyboard accessible.

## TypeScript Support

Many UI frameworks and libraries, including React, Angular, and Vue, have built-in TypeScript support, which can be useful when implementing the Badge component. Using TypeScript can help catch errors early in the development process and make it easier to maintain and scale the component over time.

Common issues when implementing the Badge component can include incorrect positioning, incorrect styling, or issues with accessibility. Troubleshooting these issues may involve inspecting the HTML and CSS, checking for errors in the code, and testing with different devices and browsers.

The Badge component is a useful UI element for highlighting important or urgent information, and can be implemented using HTML and CSS or with a UI framework or library.
