### Width

The Paper component provides a convenient width prop for controlling its horizontal dimensions. This prop accepts specific string values that correspond to predefined width classes.
**Available Width Classes:**

| Class Name Description                                                                |
| ------------------------------------------------------------------------------------- |
| `30-char` Sets the width to a minimum of 30 characters or 100%, whichever is smaller. |
| `25c` Sets the width to a minimum of 45 characters or 100%, whichever is smaller.     |
| `60c` Sets the width to a minimum of 60 characters or 100%, whichever is smaller.     |
| `75c` Sets the width to a minimum of 75 characters or 100%, whichever is smaller.     |
| `90-char` Sets the width to a minimum of 90 characters or 100%, whichever is smaller. |
| `half` Sets the width to 50%.                                                         |
| `one-third` Sets the width to 33.3%.                                                  |
| `full` Sets the width to 100%.                                                        |
| `fit-content` Sets the width to the content's intrinsic width.                        |
