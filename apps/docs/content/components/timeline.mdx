### Size

- Type: string
- Default: `md`
- Options: `2xs` | `xs` | `sm` | `md` | `lg` | `xl` | `2xl`

The size prop determines the size of the Timeline. It accepts any of the Size types defined in your library, including (3xs, 2xs,xs, sm, md, and lg, xl, 2xl, 3xl). If not provided, it defaults to the md size.

## Use Case

Timelines are often used to display a list of events in chronological order. They can be used to track progress, provide a history of events, or simply to present information in a visually appealing way.

## Implementation Details

The timeline component is implemented using HTML, CSS, and React. The HTML code for the timeline consists of a list element, which contains the events. The CSS code for the timeline defines the style of the timeline, including the color, size, and shape. The React code for the timeline renders the HTML and CSS code, and also handles events such as clicking on an event.

## Accessibility

The timeline component is accessible to all users, including those with disabilities. The HTML and CSS code for the timeline is semantic, which means that it can be read by screen readers. The React code for the timeline also includes accessibility features such as keyboard navigation and aria labels.

The timeline component is fully supported by TypeScript. The TypeScript interface for the timeline component is included in the documentation.
