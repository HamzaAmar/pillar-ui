## Props

### Size

- Type: string
- Default: `md`
- Options: `2xs` | `xs` | `sm` | `md` | `lg` | `xl` | `2xl`

The size prop controls the size of the heading. The following values are valid:

Example:

<HeadingSizes />

```tsx showLineNumbers title="Heading.tsx"
<Heading size="xs">Some heading.</Heading>
<Heading size="sm">Some heading.</Heading>
<Heading size="md">Some heading.</Heading>
<Heading size="lg">Some heading.</Heading>
<Heading size="xl">Some heading.</Heading>
<Heading size="2xl">Some heading.</Heading>
```

### Weight

The weight prop controls the weight (boldness) of the heading. The following values are valid:

Example:

<HeadingWeight />

```tsx showLineNumbers title="Heading.tsx"
<Heading weight="thin">Some heading.</Heading>
<Heading weight="extra-light">Some heading.</Heading>
<Heading weight="normal">Some heading.</Heading>
<Heading weight="bold">Some heading.</Heading>
<Heading weight="extra-bold">Some heading.</Heading>
```

### Truncate

The truncate prop with the value of "multiline" will truncate the heading if it exceeds the specified numberLine prop value and will display an ellipsis at the end of the last visible line. On the other hand, if the truncate prop value is "singleline", it will truncate the heading to fit within a single line and display an ellipsis at the end of the truncated heading.

truncate (optional): Determines whether the heading should be truncated and how it should be truncated. If the value is "singleline", the heading will be truncated to fit within a single line and display an ellipsis at the end of the truncated heading. If the value is "multiline", the heading will be truncated if it exceeds the specified numberLine prop value and display an ellipsis at the end of the last visible line.

### Align

This property specifies the horizontal alignment of the heading within the parent element. It can take one of the following values:

- start: The heading is aligned to the left (default).
- center: The heading is centered horizontally.
- end: The heading is aligned to the right.

<HeadingAlign />

### Color

- Type: string
- Default: `primary`
- Options: `danger` | `warning` | `success` | `primary` | `secondary` | `bg`

The color prop sets the color of the heading . The contrast prop adjusts the contrast between heading text and its background. The default ensures high contrast for optimal readability. Using the low prop reduces contrast, which can impact accessibility for some users.

### Leading

you can customize the leading (line height) of your heading by using the leading prop. This prop accepts four different values: "normal", "sm", "md", and "lg"

Example

<HeadingLeading />

```tsx showLineNumbers title="Heading.tsx"
<Heading leading="normal">
  This heading has a normal line height.
</Heading>
<Heading leading="sm">
  This heading has a small line height.
</Heading>
<Heading leading="md">
  This heading has a medium line height.
</Heading>
<Heading leading="lg">
  This heading has a large line height.
</Heading>

```

By using the leading prop, you can easily adjust the line height of your heading to create a more visually appealing design.

### dir RTL language

"dir": This prop sets the direction of the heading within the component. It accepts the values "rtl" for right-to-left direction or "ltr" for left-to-right direction. This prop is useful when displaying heading in languages that are read from right to left, such as Arabic or Hebrew.

<HeadingRTL />

```tsx showLineNumbers title="Heading.tsx"
export function HeadingRTL() {
  return (
    <div className="l_flow">
      <Heading dir="rtl">
        مُرَاكُش اموراكوش، «أرض الله») تسمى أيضًا بالمدينة الحمراء وعاصمة النخيل، هي ثالث أكبر مدينة في المملكة المغربية
        من ناحية عدد السكان. وهي عاصمة جهة مراكش آسفي. تقع مراكش على بعد 580 كم (360 ميل) جنوب شرق طنجة، 327 كم (203
        ميل) جنوب شرق العاصمة المغربية الرباط، 239 كم (149 ميل) جنوب شرق الدار البيضاء، و 246 كم (153 ميل) شمال شرق
        أكادير.
      </Heading>
      <Heading dir="rtl">ללמוד לקוד זה אחד הדברים הכי טובים שאני עושה כל חיי</Heading>
      <Heading dir="rtl">ކޯޑް ކުރަން ދަސްކުރުމަކީ އަހަރެންގެ މުޅި އުމުރުގައި ވެސް ކުރި އެންމެ ރަނގަޅު ކަމެކެވެ</Heading>
```
