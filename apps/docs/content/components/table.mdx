## Props

The following are the props that can be used with the Table component:

The data prop is used to provide the table data in an array format. Each element in the array represents a row in the table, and the properties of each element represent the columns.

## Use Case

The Table component can be used to display tabular data in various applications such as data grids, reports, or dashboards. It provides a structured and organized way to present data to users.

## Implementation Details

The Table component is implemented using HTML tables and styled with CSS. It supports features like sorting, striping, compactness, and borders to enhance the visual appearance and functionality of the table.

## Accessibility

The Table component is designed to be accessible and meets the following criteria:

- The table has a proper structure with header cells and data cells.
- The table has appropriate labels or headers for assistive technologies.
- The table can be navigated using keyboard controls.

## TypeScript Support

The Table component has TypeScript support and includes type definitions for its props.
