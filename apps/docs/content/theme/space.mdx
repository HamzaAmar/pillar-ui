# CSS Spacing Variables

> :warning: **Caution:** Customizing spacing using the `--space-*` variables may impact the responsive layout. Default values utilize the `clamp` function to adapt spacing based on viewport size, maintaining proportions and alignment. Modifying these values might result in spacing inconsistencies.

## Customizing the Spacing

To customize the spacing, you can override the default values of the spacing variables by assigning new values to the corresponding --space-\* variables. For example:

```css showLineNumbers title="themes.css"
:root {
  /* To customize the spacing, you can assign new values to the --space-* variables. */
  --space-md-custom: 20px; /* Customize the spacing for the md breakpoint */
  --space-lg-custom: 30px; /* Customize the spacing for the lg breakpoint */
  /* ETC */
}
```

By updating these spacing variables, you can achieve a unique layout and spacing style within your project.

## Best Practices

- **Use consistent spacing:** We recommend using the provided spacing values and leveraging the custom properties within the existing components to maintain design consistency and alignment.
- **Responsive design:** Be cautious when adjusting spacing, as it may affect the responsive layout. The default values adapt well to various screen sizes, ensuring harmonious proportions.
